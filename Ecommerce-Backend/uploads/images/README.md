# Product Images Folder

This folder contains product images for the e-commerce application.

## Required Images

The following images are required for the predefined products in data1.sql:

| Product Name | Image Filename |
|--------------|----------------|
| Tata Nexon | tata-nexon.jpg |
| Maruti Suzuki Swift | maruti-swift.jpg |
| Hyundai Creta | hyundai-creta.jpg |
| Mahindra Thar | mahindra-thar.jpg |
| Honda City | honda-city.jpg |

## Image Requirements

- Recommended size: 800x600 pixels or similar aspect ratio
- Supported formats: JPEG, PNG, JPG
- File size: Recommended max 2MB per image

## How to Add Images

1. Download or create product images
2. Save them in this folder with the exact filenames listed above
3. Restart the application

The application will automatically serve these images when displaying products.
